How to Prepare — Strategy Guide

Week 1 — Docker Foundations
  • Master all 50 Easy questions first — these are guaranteed in every interview
  • Practice writing Dockerfiles from scratch (multi-stage, non-root user)
  • Run docker-compose locally with a Spring Boot + Postgres setup
  • Understand CMD vs ENTRYPOINT, COPY vs ADD, ARG vs ENV deeply
  • Learn to debug: docker logs, docker inspect, docker exec
Week 2 — Kubernetes Core
  • Draw the K8s architecture from memory (control plane + worker node)
  • Write and apply Deployment + Service + Ingress YAMLs without docs
  • Practice kubectl: get, describe, logs, exec, rollout, scale
  • Understand ConfigMap, Secret, PVC — know when to use each
  • Practice debugging: CrashLoopBackOff, Pending, OOMKilled
Week 3 — Jenkins CI/CD
  • Write a full Jenkinsfile from scratch: Build → Test → Docker → Deploy
  • Understand declarative vs scripted pipeline differences
  • Practice parallel stages, when conditions, withCredentials
  • Learn shared library structure (vars/, src/, resources/)
  • Know the Groovy CPS trap — why some code must be in script{} blocks
Week 4 — Hard + Scenario Questions
  • Go through all Scenario questions — these test real-world thinking
  • Practice explaining your answers out loud (mock interview style)
  • Know the Tricky questions — interviewers love edge cases
  • Review the full pipeline: code → JAR → Docker → ECR → EKS
  • Practice on Katacoda, KillerCoda, or a local minikube cluster

Most Asked — Top 15 Questions

These questions come up in almost every DevOps interview. Make sure you can answer them perfectly.

Easy Docker

Container vs VM — what is the difference?

VMs virtualize hardware; containers virtualize the OS. Containers share the host kernel → 10x faster startup, 10x smaller size.

Medium Docker

How does Docker layer caching work?

Each instruction creates a layer. If instruction + context unchanged → cache hit. Order matters: COPY pom.xml before COPY src/.

Easy Kubernetes

What is Kubernetes and why use it?

Container orchestrator. Auto-restarts crashed pods, auto-scales, load balances, does zero-downtime deploys. Solves the "who manages 100 containers?" problem.

Medium Kubernetes

Deployment vs StatefulSet — when to use each?

Deployment = stateless apps (any pod is identical). StatefulSet = stateful apps needing stable hostname + ordered scaling (databases, Kafka).

Scenario Kubernetes

Pod is in CrashLoopBackOff. How do you debug?

kubectl logs pod-name --previous → kubectl describe pod → check Events section → check image, env vars, resource limits.

Easy Jenkins

What is CI/CD and why is it important?

CI = auto build+test on every commit. CD = auto deploy to staging/prod. Reduces integration hell, enables multiple deploys per day, catches bugs early.

Medium Jenkins

Declarative vs Scripted Pipeline — difference?

Declarative: structured syntax, validated, easier to read. Scripted: full Groovy, more flexible but harder to maintain. Use declarative by default.

Hard Jenkins

What is a Jenkins Shared Library?

Groovy code in a separate Git repo (vars/, src/) that multiple Jenkinsfiles can import. One place to define common steps like buildAndPush(), deployToK8s().

Tricky Jenkins

Why does withCredentials not mask secrets in double-quoted strings?

Groovy string interpolation resolves ${MY_SECRET} before Jenkins masks it. Always use single quotes: sh 'echo $MY_SECRET' not sh "echo ${MY_SECRET}".

Medium Docker

What is a multi-stage Docker build?

Multiple FROM stages in one Dockerfile. Build in stage 1 (Maven/Node) → copy only the artifact to stage 2 (JRE/nginx). Final image has no build tools → 10x smaller.

Medium Kubernetes

What is a HorizontalPodAutoscaler (HPA)?

K8s object that auto-scales pod replicas based on CPU/memory/custom metrics. Checks every 15s, scales up when metric > target, down after 5 min cool-down.

Scenario Docker

Docker image is 2GB. How do you reduce it to 200MB?

Multi-stage build, switch to alpine/JRE-slim, combine RUN commands, add .dockerignore, remove dev dependencies, avoid installing unnecessary packages.

Tricky Kubernetes

If you delete a pod managed by a Deployment, what happens?

The Deployment's ReplicaSet controller immediately creates a new pod to maintain desired replica count. The pod is replaced, not gone permanently.

Hard Docker

What is the PID 1 problem in Docker?

Docker's CMD becomes PID 1. It must handle SIGTERM for graceful shutdown AND reap zombie processes. Use exec form CMD + implement signal handler, or use --init flag.

Scenario Jenkins

Jenkins build takes 45 minutes. How do you speed it up?

Run tests in parallel, use Docker layer cache, cache Maven ~/.m2 across builds, use incremental builds, skip redundant stages on feature branches.
